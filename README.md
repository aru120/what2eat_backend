# What2eat Backend

This is the backend for what2eat. It runs on Rails and PostgreSQL

To run it, you will need a YELP API key.

--Adding YELP API key
Create .env file and put the following in the file:

YELP_KEY = "Your API KEY"

--

Run 'bundle install'
Run 'rails db:create'
Run 'rails db:migrate'

Run 'rails start'
